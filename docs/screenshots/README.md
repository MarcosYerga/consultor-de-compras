# Capturas para el README / portafolio

Aquí puedes guardar **imágenes** que muestren la app en acción (mejor que solo código en la primera impresión).

## Recomendado

1. Arranca la app (`npm run dev`), ejecuta una comparación con varias líneas y haz una captura de la **tabla de resultados** (scroll horizontal si hace falta).
2. Guarda el archivo como **`tabla-resultados.png`** en esta carpeta (PNG o WebP, ~1200px de ancho si puedes).
3. En el README principal, puedes enlazar la imagen así:

```markdown
![Tabla de resultados de comparación](./docs/screenshots/tabla-resultados.png)
```

4. Opcional: una segunda captura del **formulario** con la lista de ejemplo (`lista-compra.png`).

Si no subes aún la imagen, el README puede enlazar solo la sección “Demo en vivo”; cuando tengas PNG, añade la línea anterior al `README.md` raíz.
